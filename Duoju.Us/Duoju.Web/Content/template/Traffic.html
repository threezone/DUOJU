自驾车

  • 从广州出发（距离12Km,全程约2.5小时）：
  • 经广佛高速 → 佛开高速 → 塘口出口左转 → 开平立园碉楼 → 325国道往湛江方向 → 蚬岗镇 → 赤水镇 → 康桥温泉。
  • 从香港、深圳出发 （距离10Km,全程约3.5小时）：
  • 经广深高速 → 虎门大桥 → 京珠高速中山港口镇方向 → 中江高速共和收费站 → 佛开高速台山开平方向 → 塘口出口 左转 → 开平立园碉楼 → 325国道往湛江方向 → 蚬岗镇 → 赤水镇 → 康桥温泉。
